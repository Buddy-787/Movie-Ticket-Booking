# 🎬 Movie Ticket Reservation System

## 📌 Project Description
The Movie Ticket Reservation System is a web-based application that allows users to browse movies, select show timings, choose seats, and book movie tickets online. The system provides a simple and user-friendly interface that simulates a real-world cinema ticket booking platform.

## 🚀 Features
- View available movies and show timings
- Select seats for a movie show
- Book movie tickets online
- User-friendly interface
- Ticket booking confirmation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

cd movie-ticket-booking  

3. Open **index.html** in your browser.
